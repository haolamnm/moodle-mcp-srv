# Roadmap

`moodle-mcp-srv` is pre-1.0. Tool names and response schemas may change before stable release.

## v0.1.x

- Moodle compatibility profiles for common student and teacher Web Service setups.
- Bug fixes from real Moodle sites.
- Lean docs for installation, troubleshooting, and client configuration.

## v0.2.x

- Broader write support with consistent dry-run receipts.
- Better permission-aware feature UX without treating it as authorization.
- More graceful degradation for missing Moodle Web Service functions.

## v0.3.x

- Richer quiz, forum, assignment, and calendar behavior.
- More read-only `moodle://` resources for agent context.
- Deterministic agent-facing evaluations for common Moodle workflows.

## v1.0

- Stable MCP tool contracts and response schemas.
- Documented Moodle compatibility baseline.
- Proven release process with trusted publishing, attestations, and release checklist discipline.
