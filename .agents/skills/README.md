# Agent Skills

Project-native workflows for `moodle-mcp-srv`. Rules in `.agents/rules` are always-on guidance; skills are invoked when a task needs a specific process.

## Skills

- `codebase-design` - shared architecture vocabulary for deep modules and seams.
- `domain-modeling` - sharpen Moodle MCP terminology and update `CONTEXT.md`.
- `adr` - record durable architectural decisions in `docs/adr`.
- `tdd` - build behavior one test at a time.
- `diagnosing-bugs` - reproduce, minimize, hypothesize, fix, and verify bugs.
- `implement` - execute a scoped plan through tests and checks.
- `improve-codebase-architecture` - audit architectural friction and propose refactor candidates.
- `triage` - turn issue-like requests into clear next states and briefs.
- `ship-pr` - ship current work through a GitHub pull request using `git` and `gh`.
- `release` - cut a release with changelog, tag, checks, and GitHub release flow.
